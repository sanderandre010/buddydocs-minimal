How AI in Buddy Docs works

This document will show you how Buddy Docs works, like for example the Face ID, the one you are using.

Face ID

This app uses Tensorflow.JS for the AI, including the Face ID feature too.

How it works, is that we've trained it up on backgrounds, pets, common objects, and most importantly; faces. We then put all that data into a class, before training it, making so whenever a face is detected, it unlocks the screen. Though it doesn't unlock on a single face yet, we are developing it to manage to find out if it's a specific person or not.
